DeviceFingerprint Profiler Demo
Nov 1, 2016

Instructions:

1. Rename file cybs_devicefingerprint.js_ to cybs_devicefingerprint.js
2. Modify file devicefigerprint.html in line 12 with:
	cybersource_mid = CyberSource MID
	test/live = environment used
3. Execute file devicefigerprint.html, copy value of "Session Id" and send it into your CyberSource request
